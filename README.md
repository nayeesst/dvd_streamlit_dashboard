# DVD Rental Analytics Dashboard

This project uses Streamlit, PostgreSQL, Machine Learning, and AI features to analyze DVD rental business performance.

Features:
- Overview Dashboard
- Actor Analysis
- Country Analysis
- Rental Frequency Analysis
- Movie Success Prediction

Technologies:
- Python
- Streamlit
- PostgreSQL
- Plotly
- Scikit Learn
